# Prototype PWA hors ligne

Ce dossier contient une démonstration minimale de carnet de terrain pour smartphone.

## Fonctions
- saisie manuelle locale
- fonctionnement hors ligne après premier chargement
- installation sur écran d'accueil
- export CSV et JSON
- statut de synchronisation simulé

## Test rapide
1. lancer un serveur web statique dans ce dossier ou dans le répertoire parent
2. ouvrir la page dans le navigateur du smartphone
3. ajouter à l'écran d'accueil
4. couper le réseau pour vérifier le mode hors ligne

## Intégration future avec ESP32
Le modèle peut ensuite être relié à un point d'accès ESP32 pour envoyer les fiches locales vers l'appareil lors d'une synchronisation.
